# BBQ-advies teksten

De app toont per status (JA / TWIJFEL / NEE) en per seizoen een korte zin onder het grote woord. Ze rouleren random, dus je ziet niet elke keer dezelfde.

**Jouw taak:** maak de rechterkolommen leuker / droger / knipogender. Elke regel mag vrij veranderd, nieuwe regels toevoegen mag ook.

---
